# Slim Fit
Slim Fit is a 8pt pixel art font designed to have sharp tall lettering, with rounded variations.

## Author
[VEXED](https://v3x3d.itch.io/)

## Basic Info
- Base Glyph Height: 8px
- Scales: 8, 16, 24, 32, etc
- Ascender Max: 3px
- Descender Max: 1px

## Kerning
None
